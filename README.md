# Programación Orientada a Objetos 💻
Este repositorio contiene los trabajos prácticos, ejercicios y proyectos desarrollados para la cátedra de **Programación Orientada a Objetos** de la carrera de **Ingeniería de Sistemas**.

## 🎯 Objetivos del Repositorio
El propósito es documentar el aprendizaje sobre el paradigma orientado a objetos, aplicando buenas prácticas y diseño de software.

## 📚 Contenidos Principales
A lo largo del curso, se verán implementaciones sobre:
*   **Pilares de POO:** Abstracción, Encapsulamiento, Herencia y Polimorfismo.
*   **Gestión de Memoria:** Manejo de objetos y referencias.
*   **Relaciones:** Asociación, Agregación y Composición.
*   **Interfaces y Clases Abstractas.**
*   **Manejo de Excepciones.**
